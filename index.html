About Project Caelum



The main objective of Project Caelum is to create a fun and interactive data visualization website to raise awareness about the global climate change.

Project Caelum uses viewer-driven narrative approach of data visualization which allow users to freely interact with the data and tell their own story. Live, real time data would be automatically updated every hour. However, This project is still in alpha version with lack of many interactive function.
